# Quickstartpygame.py

## Introduction
This is a simple Pygame program that demonstrates how to create a window and draw a rectangle.

## Requirements
- Python 3.x
- Pygame

## Installation
1. Clone the repository.
2. Install Pygame by running `pip install pygame` in your terminal.
3. Run the program by typing `python quickstartpygame.py` in your terminal.

## Usage
- Press the `Esc` key to exit the program.
- Move the rectangle using the arrow keys.
